# Example 项目介绍

### 项目概述

ttswap是一个建立在支持EVM区块链上的自动做市协议，意味着它不依赖于中心化的机构或个人来进行交易。它的核心原理是根据用户的行为，自动触发市场价值的转移，从而构建了一个基于恒定价值交易模型的协议。 

### 参加赛道
主赛道：xFi
Bounty 赛道：TheGraph

### 核心功能

1、交易无中转无滑点
2、投资(流动性提供)没有无常损失
3、按角色分佣
4、低GAS费
